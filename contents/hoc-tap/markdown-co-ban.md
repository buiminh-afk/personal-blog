# Markdown Cơ Bản - Hướng Dẫn Toàn Tập

Markdown là một ngôn ngữ đánh dấu văn bản đơn giản, dễ học và được sử dụng rộng rãi trong việc viết tài liệu, blog, và README files. Trong bài viết này, chúng ta sẽ tìm hiểu về các cú pháp Markdown cơ bản và cách sử dụng chúng hiệu quả.

## Tại Sao Nên Học Markdown?

Markdown mang lại nhiều lợi ích cho người viết:

- **Đơn giản và dễ học**: Cú pháp rất trực quan, bạn có thể học trong vài phút
- **Tập trung vào nội dung**: Không bị phân tâm bởi các công cụ định dạng phức tạp
- **Tương thích cao**: Được hỗ trợ trên nhiều nền tảng như GitHub, Reddit, Discord
- **Dễ chuyển đổi**: Có thể chuyển sang HTML, PDF, hoặc các định dạng khác

## Cú Pháp Cơ Bản

### Tiêu Đề (Headings)

Sử dụng dấu `#` để tạo tiêu đề. Số lượng dấu `#` quyết định cấp độ của tiêu đề:

```markdown
# Tiêu đề cấp 1
## Tiêu đề cấp 2
### Tiêu đề cấp 3
```

### Định Dạng Văn Bản

- **In đậm**: Sử dụng `**text**` hoặc `__text__`
- *In nghiêng*: Sử dụng `*text*` hoặc `_text_`
- ~~Gạch ngang~~: Sử dụng `~~text~~`
- `Code inline`: Sử dụng backticks

### Danh Sách

Danh sách không thứ tự:
```markdown
- Item 1
- Item 2
  - Sub item
```

Danh sách có thứ tự:
```markdown
1. First item
2. Second item
3. Third item
```

## Code Blocks

Để hiển thị code, sử dụng ba backticks:

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

## Links và Images

Tạo link: `[Text hiển thị](URL)`

Chèn ảnh: `![Alt text](image-url)`

## Kết Luận

Markdown là công cụ tuyệt vời cho việc viết và chia sẻ nội dung. Với cú pháp đơn giản nhưng mạnh mẽ, bạn có thể tạo ra các tài liệu chuyên nghiệp mà không cần phải lo lắng về định dạng phức tạp.

Hãy thực hành thường xuyên để thành thạo Markdown nhé!
